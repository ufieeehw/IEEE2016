ieee2015_msgs
=============


This package contains ROS message types specific to the uf ieee2015 robot, Maverick.

# Manifest
## Mecanum.msg
Four float32 elements, vel_0...vel_3, each reflecting the rotational velocity in rad/s of one of the four mecanum wheels

    float32 vel_0
    float32 vel_1
    float32 vel_2
    float32 vel_3
